<p>These design activities are meant to help designers engage with the SERG framework, either guiding a process of reflection or creation. We have provided several activities that may serve useful regardless of stage in the game design process. This, alongside the next section “Goal Setting” should be used as a set of tools to engage with the framework. Underlying any game using such should be clear and intentional choices around how a game might engage concepts of race as well as real-world impacted communities.</p>
