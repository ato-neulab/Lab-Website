<p>Different members of a game studio can benefit from this framework differently. For example, game designers may consider what aspects of a game they would like to be implemented and developers can consider directly how to go about creating a given feature. Someone on the marketing team, however, could use the framework as a guide on which aspects of the game may be most salient to audiences. <br><br>
As such, we encourage studio members of different levels and positions to use this framework as a communicative tool to discuss the detailed and overarching strengths and weaknesses of a game. For example, if a game is placed lower than where the design team had wished, this can bring about conversations as to if hiring a cultural consultant is necessary.</p>
